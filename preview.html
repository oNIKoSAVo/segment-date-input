Web Component · segmented date · ISO value

Техническая проверка кастомного date input

Компонент отображает дату сегментами дд/мм/гггг, принимает ввод с клавиатуры, хранит значение в ISO-формате yyyy-mm-dd и отправляет события input / change.

Custom element <segment-date-input>
Displayed format дд/мм/гггг
Value format YYYY-MM-DD

Usage

							
Библиотека регистрирует web component через side-effect import.

						

					


					

						

							
Import

							app entry
						

						
import "segment-date-input";

					


					

						

							
HTML

							custom element
						

						
<segment-date-input
  value="2026-04-26"
  min="2000-01-01"
  max="2026-12-31"
  placeholder="Дата"
  title="Дата"
></segment-date-input>

					

				


				

					

						

							
SolidJS example

							
Значение хранится в ISO-формате и обновляется через событие input.

						

					


					

						

							
TSX

							controlled value
						

						
<segment-date-input
  value={props.value}
  min={min()}
  max={max()}
  placeholder={props.label}
  title={props.label}
  disabled={props.disabled}
  onInput={(ev) => {
    const value = ev.currentTarget.value;
    props.setValue(value);
  }}
/>

					

				

			


			

				

					

						
Attributes / properties

						
Атрибуты синхронизируются с одноимёнными свойствами компонента.

					

				


				

					
						

							Name
							Type
							Default
							Description
						

					
					
						

							value
							string
							""
							Текущее значение даты в формате YYYY-MM-DD.
						

						

							min
							string
							2000-01-01
							Минимально допустимая дата. Применяется при нормализации значения.
						

						

							max
							string
							current date
							Максимально допустимая дата. По умолчанию используется текущая дата.
						

						

							disabled
							boolean
							false
							Блокирует ввод и фокусируемое поведение внутреннего input.
						

						

							placeholder
							string
							""
							Передаётся во внутренний input.
						

						

							title
							string
							""
							Передаётся во внутренний input.
						

					
				

			


			

				

					

						

							
Methods

							
Публичные методы доступны через DOM-ссылку на элемент.

						

					


					

						
							

								Method
								Description
							

						
						
							

								focus()
								Фокусирует внутренний input.
							

							

								selectSegment(segment)
								Выделяет сегмент day, month или year.
							

							

								clear()
								Очищает значение, сбрасывает draft и отправляет события.
							

						
					

				


				

					

						

							
Styling

							
Внутренний input открыт через part="input-date".

						

					


					

						

							
CSS

							::part
						

						
segment-date-input::part(input-date) {
  min-height: 48px;
  padding: 0 14px;
  border: 1px solid #bbc9b7;
  border-radius: 13px;
}

segment-date-input::part(input-date):focus {
  border-color: #24723c;
  box-shadow: 0 0 0 4px rgb(36 114 60 / 12%);
}

Event payload

							
Компонент отправляет CustomEvent с detail.value.

						

					


					

						

							
Listener

							JS
						

						
dateInput.addEventListener("input", (event) => {
  console.log(event.detail.value);
});

dateInput.addEventListener("change", (event) => {
  console.log(event.detail.value);
});
